## Installation
Github:

    git clone https://github.com/I2rys/GARTLP
    
NPM Packages:
```
npm i to-hex
```

Test NPM Packages:
```
npm i simple-exec
```

## Usage
```
node index.js <link>
```
+ link - Your link to embed to the payload.

## Dev Usage
NPM test:
